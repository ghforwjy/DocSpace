## 十、技术栈参考

### 后端技术栈
- **语言**: C# 11.0
- **框架**: .NET 7.0 / .NET Core / ASP.NET Core
- **数据库**: MySQL 8.0
- **消息队列**: RabbitMQ / Kafka
- **搜索引擎**: ElasticSearch / OpenSearch
- **缓存**: Redis

### 前端技术栈
- **语言**: TypeScript / ES6
- **框架**: React
- **状态管理**: MobX
- **样式**: Styled-Components / SCSS / CSS
- **国际化**: i18next
- **构建工具**: Webpack 5 / pnpm
